Basics of CSS

Common Selectors

Targets an HTML element

  • Element Selector: Targets all instances of an element.
  • Class Selector:Targets elements with a specific class.
  • ID Selector: Targets a specific element with an ID

Property

Defines what to style.

Value

Specifies the styling applied.

Basic Styling Properies

Colors

Style elements with background and text colors.

Font Styles

Customize text appearance.

Text Alignment

Control text placement within an element.

Spacing with Padding and Margin

  1. Padding: Space inside the element
  2. Margin: Space outside the element

Box Model

Margin

Outer space between the element and others

Padding

Space between the content and the border.

Border

Surrounds the padding.

Content

The inner part containing text or other elements.
